# Mystery-Mansion-Game
Text adventure game for a project.
In MYSTERY MANSION, you explore an abonded building filled with clues, 
ghostly rooms and airy descriptions. 
Your goal is to figure out the secret the mansion has hidden for decades
and find a way out - or can you....

How to play
- Run the game.py file in python.
- Type commands e.g.
  GO NORTH
  GO EAST
  TAKE NOTE
  READ MESSAGE
- Explore rooms, figure out clues and solve the hidden secret.

Game Overview

You start in the Grandhall in the dark mansion. Cobwebs cover your visions,
odd notes appear out of nowhere, and each movements goes deeper in the story.
Secret communications and items will help you move on the game - but you got 
to choose wisely....

Features

- Fully text based
- Many rooms and navigations
- Secret messages and item interaction
- Eerily storytelling

Requirements
- Python 3.x

Files 
- game.py - main game file
- READ.md - game description
